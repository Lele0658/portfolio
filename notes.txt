# Your Personal Website: A Complete Beginner's Guide

I'll explain everything step by step so you can understand how your website works!

## What is a Website?

Think of a website like a **digital house**:
- **HTML** = The structure (walls, rooms, doors)
- **CSS** = The decoration (paint, furniture, colors)
- **JavaScript** = The electricity (lights that turn on/off, interactive features)

## Let's Break Down Your Website Piece by Piece

### 1. The Basic Structure (HTML)

```html
<!DOCTYPE html>
<html>
<head>
    <!-- This is like the "blueprint" of your house -->
    <title>Your Name | Portfolio & Blog</title>
</head>
<body>
    <!-- This is where all the visible content goes -->
    <header>Navigation Menu</header>
    <section>Home Section</section>
    <section>About Me</section>
    <!-- etc... -->
</body>
</html>
```

### 2. Your Website Sections Explained

**🏠 Home Section** - The front door of your digital house
- Big welcome message with your name
- Your photo/avatar
- Quick introduction
- Buttons to see your work or contact you

**👤 About Me** - Your personal story room
- Who you are and what you do
- Your background and interests
- Your hobbies (shown as little tags)

**💼 Portfolio** - Your work showcase room
- Displays your projects like pictures on a wall
- Each project has:
  - Project image
  - Description
  - Technology tags (like "HTML", "JavaScript")
  - Link to view the project

**🛠️ Skills** - Your tools and abilities display
- Shows what you're good at
- Visual progress bars for each skill
- Organized by categories (Frontend, Backend, Tools)

**📝 Blog** - Your thoughts and writings
- Blog posts displayed as cards
- Each has title, date, category, and excerpt
- "Read More" button for full articles

**🖼️ Gallery** - Your personal photos space
- Grid of images that represent your life/interests
- Hover effects for interactivity

**📞 Contact** - Your communication center
- Contact information (email, phone, location)
- Social media links
- Contact form for visitors to message you

### 3. The Color System

Your website uses these colors:
- **Pink** (#ff4da6) - For accents and highlights
- **Blue** (#3f8efc) - For links and some backgrounds
- **Purple** (#a259ff) - For buttons and important elements
- **Black** (#111111) - For dark mode background

### 4. Light/Dark Mode Magic

This is like having **light switches** in your house:

```javascript
// When you click the theme button:
themeToggle.addEventListener('click', () => {
    if (current theme is light) {
        change to dark theme
        save preference
    } else {
        change to light theme
        save preference
    }
});
```

The website remembers your choice using `localStorage` - like writing your preference in a notebook that the browser remembers.

### 5. How to Customize Your Website

**Easy Changes You Can Make:**

1. **Change Your Name**:
   - Find: `<div class="logo">Your<span>Name</span></div>`
   - Replace "YourName" with your actual name

2. **Update Contact Info**:
   - Find the contact section
   - Change email, phone, and location

3. **Add Your Real Projects**:
   - In the portfolio section, replace the placeholder projects
   - Add real project images, descriptions, and links

4. **Customize Skills**:
   - Update the skill names and percentages
   - Add or remove skills as needed

### 6. How the Website Responds to Different Screens

Your website automatically adjusts for:
- **Computers** (shows full navigation menu)
- **Tablets** (adjusts layout but keeps most features)
- **Phones** (shows hamburger menu ☰ and stacked content)

    <script>
        // THEME TOGGLE FUNCTIONALITY
        
        // Get the theme toggle button and body element
        const themeToggle = document.getElementById('themeToggle');
        const body = document.body;
        
        // Check if user previously set a theme preference
        const currentTheme = localStorage.getItem('theme') || 'light';
        
        // Apply the saved theme
        body.setAttribute('data-theme', currentTheme);
        
        // Update button emoji based on current theme
        if (currentTheme === 'dark') {
            themeToggle.textContent = '☀️';  // Sun for light mode
        } else {
            themeToggle.textContent = '🌙';  // Moon for dark mode
        }
        
        // When theme button is clicked
        themeToggle.addEventListener('click', () => {
            if (body.getAttribute('data-theme') === 'light') {
                // Switch to dark theme
                body.setAttribute('data-theme', 'dark');
                themeToggle.textContent = '☀️';
                localStorage.setItem('theme', 'dark');  // Save preference
            } else {
                // Switch to light theme
                body.setAttribute('data-theme', 'light');
                themeToggle.textContent = '🌙';
                localStorage.setItem('theme', 'light');  // Save preference
            }
        });
        
        // MOBILE MENU TOGGLE
        
        const mobileMenu = document.querySelector('.mobile-menu');
        const navLinks = document.querySelector('.nav-links');
        
        // When hamburger menu is clicked
        mobileMenu.addEventListener('click', () => {
            // Toggle between showing and hiding the menu
            navLinks.style.display = navLinks.style.display === 'flex' ? 'none' : 'flex';
        });
        
        // CONTACT FORM SUBMISSION
        
        const contactForm = document.getElementById('contactForm');
        
        // When form is submitted
        contactForm.addEventListener('submit', (e) => {
            e.preventDefault();  // Prevents page reload
            
            // Show success message
            alert('Thank you for your message! I will get back to you soon.');
            
            // Clear the form
            contactForm.reset();
        });
        
        // SMOOTH SCROLLING FOR NAVIGATION LINKS
        
        // Get all links that start with # (internal links)
        document.querySelectorAll('a[href^="#"]').forEach(anchor => {
            anchor.addEventListener('click', function (e) {
                e.preventDefault();  // Prevents jumpy behavior
                
                const targetId = this.getAttribute('href');
                
                // Don't do anything if it's just #
                if (targetId === '#') return;
                
                // Find the target section
                const targetElement = document.querySelector(targetId);
                
                if (targetElement) {
                    // Smoothly scroll to the section
                    window.scrollTo({
                        top: targetElement.offsetTop - 80,  // Adjust for header height
                        behavior: 'smooth'                 // Smooth animation
                    });
                    
                    // Close mobile menu if open
                    if (window.innerWidth <= 768) {
                        navLinks.style.display = 'none';
                    }
                }
            });
        });
    </script>
